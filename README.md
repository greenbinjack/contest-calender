# Contest Calender

A dark-mode, mobile-first competitive programming contest calendar.

## Features
- **Real-Time Tracking**: Get live updates on upcoming and active contests.
- **Supported Platforms**: Codeforces, AtCoder, CodeChef, LeetCode, Google, Gym, Eolymp, LightOJ, and Kattis.
- **UI**: A beautiful, kinetic dark-mode interface built with custom CSS animations and GSAP (ScrollTrigger).
- **PWA & Mobile Install**: Fully packaged as a Progressive Web App (PWA) with a manifest and service worker, ready to be turned into a mobile APK.
- **Static Splash Screen**: Automatically displays the application's icon on opening before rendering the fetched dashboard.

## Tech Stack
- **HTML/CSS/JS**: Vanilla frontend architecture with no heavy external frameworks.
- **GSAP**: Used for floating code background particles and smooth scroll animations.
- **Clist.by API**: Fetches reliable, up-to-date competitive programming contest schedules.

## Download the App
An Android `.apk` file is included in this repository! You can download it directly to your mobile device, install it, and start tracking contests on the go without needing a browser.
